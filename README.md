# ping_test
Pingテスト用シェルスクリプト。対象サーバが動いているか確認する際にpingを実行するのに何度も 
同じコマンドを打って対象サーバを探しての繰り返しは面倒なので、リスト化した対象サーバに対して
pingを実行するようなスクリプトを作成。

## 機能(Feature)
ping実行用のサーバリストを準備し、その対象に対してpingを実行。実行結果をログに残す。
